# projet-ecommerce-react
Projet de site E-Commerce avec React 
Ceci n'est pas un site réel vente!!!!!!!!
Je vous présente ce projet e-commerce React, en Guest l'utilisation de commerce.js et l'outil de paiement
Stripe, parmi un choix de téléphones(les photos proviennent d'Unsplash), vous avez la possibilité de 
d'ajouter au panier vos modèles préférés, rentrer vos coordonnées de livraison
ainsi que de simuler un paiement POUR TESTER (pas de beoin de votre vrai carte, voici le num factice : 4242 4242 4242 4242 04/24 424),
pour vérifier que la commande est 'passé' renseigner votre email vous recevrez un mail de confirmation!

C'est un projet test , rien n'est réel!
